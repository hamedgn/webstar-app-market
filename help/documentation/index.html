BACKEND SETTINGS

Thank you for purchased our product! We hope you like our app!

Let's start configuring your application.

The documentation is written in an example if you are hosting with cpanel

Detailed documentation will allow you to configure the application in a short time.

The first thing you need to do is configure your backend will store all information applications.

1) Unzip the archive with the application and locate the file PHP_BACKEND.zip

2) Unzip file S_MUSIC_PHP_BACKEND.zip

3) Upload all files from the root directory of your site using your favorite FTP application.

4) Create in hosting panel new database MySql

5) Open phpMyAdmin and choose your new database 

6) Click "Import" button and choose database.sql file in application folder

7) Click "Go" and import database.

 8) Next: configure backend ( database and url )

9) Open file database.php ( application -> config folder)

10) Change the data from the server, database name, password, database user

11) Next, open file config.php ( application->config folder)

12) Change base_url, sender_email and sender_name 

13) Open your browser to the backend, you should see the login page.

14) Access Login and Password : admin

 

Next, connect the service Amazon S3 for uploads media files

AMAZON S3 SETTINGS

If you do not have an account at Amazon, click here and create your account.  https://aws.amazon.com/?nc1=h_ls

1) Login or sign up in Aws Amazon

2) In Amazon S3 console create new bucket

3) Get in Amazon console assecc key and a secret key ( How to do it, read the official documentation Amazon Aws https://aws.amazon.com/documentation/s3/?nc1=h_ls )

4) Open file Amazon3integration_lib.php in application->libraries folder

5) Change bucket name and regions ( region you can see in the bucket of properties in the control panel amazon S3 )

6) Scroll page down and change amazon access key and amazon secret key 

7) Go to Amazon S3 console

8) Choose bucket , that you have done for the backend and click on it

9) Create 4 new folders (album, artist, image, music)

If done correctly, your backend can upload files to Amazon S3

Next , settings  application

APP SETTINGS

Setting up the application will take a couple of minutes, it's easy.

1) Open the project using Android Studio

2) Find file Constant.java

3) Change strings:

BASE_BACKEND_URL - Full  url link to your backend

DEF_TRACK_SEARCH - Any word search screen

BASE_BACKEND_AMAZON_BUKET - Amazon S3 buket

DEF_IMAGE - A link to the default imagecover if don't found in Spotify

AD_INTERSTITIAL_ID - ID interstitial

AD_BANNER_ID - ID Banner

SHOW_AD - Show / hide Ads

on this, the application configuration is finished

If you do not know how to create an identity advertising, read the next chapter

 

 

ADMOB SETTINGS

This application is integrated ad serving on Google

1) Go to Google Admob service 

2) Click on "Monetize" button and "Monetize new app"

3) Choose "Add your app manually", fill App name, select platform "ios", click "Add app"

4) Select Ad format - "Banner", fill "Ad unit name" and click "Save"

5) Copy Ad unit ID 

6) Open Android project , find file Constant.java and paste ID ADs
